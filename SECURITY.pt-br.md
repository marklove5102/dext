# Política de Segurança

## Versões Suportadas

Atualmente, fornecemos atualizações de segurança para as seguintes versões:

| Versão  | Suportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reportando uma Vulnerabilidade

Levamos a segurança a sério. Se você encontrar uma vulnerabilidade de segurança no Dext Framework, por favor, relate-a através do nosso **[GitHub Issues](https://github.com/Dext/DextRepository/issues)** utilizando a label `security`.

Por favor, inclua:
*   Uma descrição da vulnerabilidade.
*   Uma prova de conceito ou passos para reproduzir.
*   Impacto potencial.

Confirmaremos o recebimento do seu relato em até 48 horas e forneceremos um cronograma para a correção, se aplicável.
