# Security Policy

## Supported Versions

We currently support and provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you find a security vulnerability in the Dext Framework, please report it via our **[GitHub Issues](https://github.com/Dext/DextRepository/issues)** using the `security` label.

Please include:
*   A description of the vulnerability.
*   A proof of concept or steps to reproduce.
*   Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix if applicable.
